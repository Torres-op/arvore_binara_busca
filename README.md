## **Projeto de Árvore Binária de Busca**

É um projeto feito para um trabalho de faculdade, aonde o objetivo era criar uma Árvore Binária de busca. Este projeto foi feito em Python utilizando as bibliotecas:
- **MatPlotLib**: Para criar gráficos e visualizações
- **NetworkX**: Para trabalhar com grafos (estruturas de nós e arestas)
- **TKInter**: Para criar a interface gráfica do Software
